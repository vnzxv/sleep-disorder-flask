# Sleep Disorder Prediction
Machine Learning web app using Gaussian Naive Bayes and K-Means Clustering.

## Stack
- Python 3.14
- Flask
- scikit-learn
- WEKA (for validation)

## How to Run
1. Install: `pip install flask scikit-learn pandas numpy`
2. Run: `python app.py`
3. Open: `http://127.0.0.1:5000`

## Results
- Naive Bayes Accuracy: 86.67% (Python) / 89.33% (WEKA)
- K-Means Clustering: 63.90%
